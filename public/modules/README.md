# 🎯 Fundamental — Fullstack Interview Preparation

Persiapan interview untuk posisi **Fullstack Developer (React + Laravel)** level **Fresh Graduate**.

## Learning Path (2-4 minggu)

### Minggu 1: Foundation
```
Day 1-2:  01-web-fundamental.md
Day 3-5:  02-javascript-deep.md
Day 6-7:  Practice soal JS + review
```

### Minggu 2: Frontend
```
Day 1-3:  03-react-fundamental.md
Day 4-5:  Build mini project React (CRUD sederhana)
Day 6-7:  Practice + review
```

### Minggu 3: Backend & Database
```
Day 1-3:  04-php-laravel.md
Day 4-5:  05-database-sql.md
Day 6-7:  Build mini API Laravel + CRUD
```

### Minggu 4: Wrap Up
```
Day 1-2:  06-cs-fundamental.md
Day 3-4:  07-fullstack-scenario.md
Day 5:    08-quick-cheatsheet.md (hafalan)
Day 6-7:  Mock interview + review
```

## Cara Pakai

1. Baca setiap file mulai dari `01` sampai `07` secara urut.
2. Setiap selesai baca satu modul, coba jelaskan ulang dengan kata sendiri.
3. Jawab pertanyaan di akhir modul (simulasi interview).
4. Praktik coding langsung (bikin project kecil).
5. H-1 interview, baca `08-quick-cheatsheet.md` buat hafalan cepat.

## Tambahan
- Latihan soal: buka [LeetCode](https://leetcode.com) bagian Easy (JS/PHP)
- Baca dokumentasi resmi: [React.dev](https://react.dev), [Laravel.com](https://laravel.com)
- Cari pertanyaan interview di [Glassdoor](https://glassdoor.com) atau [Reddit](https://reddit.com/r/cscareerquestions)
